# linux-setup-script

OOTB productive setup script for [debian, arch, fedora, nix], GNOME.

## setup

### 0. prerequisites

- python3-venv
- git

### 1. clone this repo

```bash
git clone https://github.com/seolcu/linux-setup-script
cd linux-setup-script
```

### 2. setup venv

```bash
python3 -m venv venv
source venv/bin/activate
pip3 install -r requirements.txt
```
